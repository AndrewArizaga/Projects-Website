Robot Arm on a Moving Car

This project integrates a robotic arm with a moving car platform to perform tasks requiring both mobility and precision. The system is designed for remote operation and features an embedded system that controls both the car's navigation and the arm's movements. Applications include industrial automation, delivery robots, and mobile manipulation systems.

Key Features:

  • Combines mobility with precision object manipulation.
  • Uses embedded systems for integrated control of car and robotic arm.
  • Supports remote operation for dynamic tasks.

Key Skills:

  • Robotics design and programming
  • Motor control and motion planning
  • Embedded systems and IoT integration
