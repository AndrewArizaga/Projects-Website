IoT-Based Automated Plant Watering System

This project leverages IoT technology to create an automated plant watering system that ensures optimal moisture levels. The system utilizes soil moisture and humidity sensors to monitor environmental conditions and activate an Arduino-controlled pump for precise watering. This sustainable solution simplifies plant care and offers practical applications for smart home automation.

Key Features:

  • Monitors soil moisture and humidity in real-time.
  • Activates a pump to water plants precisely when needed.
  • Offers a sustainable and efficient smart home solution.

Key Skills:

  • IoT system integration
  • Arduino programming
  • Automation and sensor-based design
