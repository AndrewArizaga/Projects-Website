Smart Power Management System
Using Microcontroller

The **Smart Power Management System Using Microcontroller** enhances energy efficiency by automating light and appliance control based on room occupancy and individual identification. This innovative system uses an ESP32 microcontroller, custom PCB design, and advanced sensors to optimize power usage.

Project Description

Overview

This project leverages modern IoT technology to develop a smart power management system. It automates energy-efficient operations by detecting room occupancy through IR sensors and identifying users via iris recognition.

Key Features

- **Automated Controls**: Adjusts devices based on occupancy and identity. - **Energy Savings**: Reduces power wastage for sustainable living. - **Custom PCB**: Designed for reliability and scalability.

System Workflow

The system processes sensor inputs to determine room occupancy and identity. Based on this data, relays are triggered to control connected devices. The workflow includes detection, recognition, and execution phases.
