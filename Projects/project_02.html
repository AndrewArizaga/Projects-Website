Project Overview

The Face and Voice Recognition Based Auto Door project is a sophisticated automation system that enhances security and convenience by integrating facial and voice recognition technologies with a motorized door mechanism. This project showcases expertise in microcontroller programming, circuit design, and hardware integration.

Project Details

Objective

To develop an automatic door system that opens only for authorized individuals using facial and voice recognition, ensuring enhanced security and reducing unnecessary activations.

Components Used

  • ESP32-CAM for facial recognition
  • Arduino Uno for processing and control
  • Serial voice recognition module
  • Stepper motor for door actuation
  • Custom PCB designed in KiCad

Features

  • Real-time face detection and recognition
  • Voice commands for access control
  • Integration of multiple hardware components on a custom PCB
  • Efficient and secure automation for home or office doors

Challenges Overcome

Ensuring accurate facial recognition in various lighting conditions, optimizing voice recognition for different accents, and achieving seamless integration of all components on the PCB were some of the key challenges successfully addressed during the project.

Outcome

The project resulted in a functional prototype that demonstrates the feasibility and effectiveness of combining face and voice recognition for automated door control, providing a foundation for further development into a commercial product.
