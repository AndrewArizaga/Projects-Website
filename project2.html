Face and Voice Recognition Auto Door

This project aimed to enhance security systems by integrating face and voice recognition technologies with IoT components. Designed for both home and office environments, the system provides secure and convenient access control. Key features include:

  • Utilized AI-driven face and voice recognition for authentication.
  • Integrated IoT modules for seamless connectivity and monitoring.
  • Developed using microcontroller programming and real-time data processing.

This innovative approach highlights the practical application of advanced AI and IoT in solving real-world challenges, enhancing both security and user convenience.

Key Skills: AI, IoT, microcontroller programming, security systems design.
