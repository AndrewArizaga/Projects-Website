Projects

Smart Power Management Using Microcontroller

This project focuses on developing a smart power management system that uses a microcontroller to monitor and optimize energy consumption. The system integrates sensors to measure electrical parameters and automates power distribution based on real-time data. By improving energy efficiency and reducing waste, this project highlights practical applications in sustainability and modern power systems.

Key Skills: Microcontroller programming, sensor integration, embedded systems design.

Face and Voice Recognition-Based Auto Door

A security-driven automation project, this system uses a microcontroller to implement face and voice recognition for controlling door access. The project integrates recognition modules and sensors to authenticate users, providing a secure and seamless automation experience. It demonstrates the potential of IoT technologies in enhancing home and workplace security systems.

Key Skills: IoT integration, microcontroller programming, AI-powered recognition systems.

Machine Learning-Based Energy Meter

This project combines IoT hardware and machine learning to create an intelligent energy monitoring system. Using an Arduino UNO, the system calculates real-time voltage, current, and power consumption while collecting data for analysis. Machine learning algorithms predict energy usage trends and detect anomalies, offering valuable insights for optimizing energy consumption.

Key Features: Real-time data display, energy analysis, intelligent predictions.

Key Skills: IoT systems, data analytics, machine learning.

Python Program for Ladder Iterative Load Flow

A foundational project in power systems, this tool uses Python programming to implement the ladder iterative method for analyzing load flow in electrical grids. The program calculates power distribution and performance metrics, providing a clear understanding of network behavior. This project is particularly useful for optimizing grid performance and planning.

Key Skills: Python programming, power system analysis, algorithm design.
