Projects

Hey there! These are some of the projects I worked on during my undergraduate degree. I pursued these projects not only to enhance my technical skills but also because I found them deeply fascinating. They reflect my curiosity and ambition, and working on them made me feel like a true engineer tackling real-world challenges.

Smart Power Management Using Microcontroller

This project focuses on designing a smart power management system using a microcontroller to monitor and control energy consumption. Sensors measure power usage in real time, enabling the system to optimize energy distribution efficiently. The project showcases practical applications in sustainability and modern power systems.

Key Skills: Microcontroller programming, sensor integration, embedded systems design.

Face and Voice Recognition Auto Door

A security-driven automation system utilizing face and voice recognition with a microcontroller to authenticate and control door access. This project integrates cutting-edge recognition modules and IoT components to enhance security and user convenience in home or office environments.

Key Skills: IoT technologies, microcontroller programming, AI-powered recognition systems.

Machine Learning-Based Energy Meter

This project integrates IoT hardware and machine learning to monitor, analyze, and predict energy consumption. Using an Arduino UNO, it measures voltage, current, and power in real time. Machine learning algorithms detect usage trends and anomalies, providing valuable insights for energy optimization.

Key Skills: IoT systems, data analytics, machine learning.

Python-Based Ladder Iterative Load Flow Tool

A fundamental project in power systems that develops a Python program to calculate load flow using the ladder iterative method. This tool visualizes power distribution and performance in electrical grids, helping optimize network efficiency and planning.

Key Skills: Python programming, power system analysis, algorithm design.

IoT-Based Automated Plant Watering System

This system uses IoT technologies to automate plant watering based on real-time humidity and soil moisture data. Sensors detect the plant's needs, and an Arduino-controlled pump waters it accordingly. The project offers practical applications for smart home automation and sustainable gardening.

Key Skills: IoT integration, sensor programming, automation systems.

Robot Arm on a Moving Car

A robotics project involving the integration of a robotic arm on a mobile platform, enabling object manipulation and mobility. The system is controlled using embedded systems and IoT for remote operation. This project demonstrates advanced robotics and mechatronics integration.

Key Skills: Robotics design, motor control, embedded systems.
