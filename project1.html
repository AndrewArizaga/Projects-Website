Smart Power Management Using Microcontroller

This project focuses on designing a smart power management system using a microcontroller to monitor and control energy consumption. It measures real-time power usage, optimizing energy distribution for sustainability and efficiency. The system highlights the integration of sensors and embedded systems for modern power solutions.

Key Features:

  • Monitors energy usage in real-time.
  • Optimizes power distribution for sustainability.
  • Demonstrates practical applications of embedded systems.

Key Skills:

  • Microcontroller programming
  • Sensor integration
  • Embedded systems design
