Machine Learning-Based Energy Meter

This project integrates IoT hardware and machine learning algorithms to monitor, analyze, and predict energy consumption. By leveraging real-time data, the system identifies usage trends and detects anomalies, providing valuable insights for optimizing energy efficiency. It demonstrates the application of advanced analytics in modern energy systems.

Key Features:

  • Real-time monitoring of voltage, current, and power usage.
  • Machine learning-based anomaly detection and trend analysis.
  • Actionable insights for energy optimization and efficiency.

Key Skills:

  • Machine learning
  • IoT hardware integration
  • Data analytics
